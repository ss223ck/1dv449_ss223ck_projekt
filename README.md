# Projektarbete webteknik II
## Mashup-applikation

Jag har valt att skriva en mashup-applikation som använder api:erna open-stockholm för att hämta möjliga ställen att parkera
på för att kombinera det med en karta. Kartarn visar vägarna man kan parker på. Det ska även gå att välja specifik fordon. Om ska även
kunna ange en addres så visas närmsta pakeringsplats. Om applikationen blir för liten kommer jag även lägga in sorteringsalternativ på 
kostnader på olika pakeringsplatser.
